# Panda-Screen_pet
An interactive Panda screen_pet in Python

Pet is inspired from panda of "We Bare Bears".

<img src="images\og.jpg" width="400">

## Modules Used
```
tkinter
pygame
```
## Features

* Blinking
* Happy (blush)
* Normal face
* Sad
* Funny (on double click)
